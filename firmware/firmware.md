# Firmware Documentation

This section provides detailed information about the firmware used in Telemetry On Tap devices, including installation instructions, configuration options, and firmware parameters. 

## Connecting and Updating Firmware
Firmware updates are done through an integrated web UI. To update firmware, you must first connect to the device and access the web UI, then follow the update process.

- [Device Connect](https://device.docs.telemetryontap.com/firmware/device_connect)
- [Update](https://device.docs.telemetryontap.com/firmware/update)
